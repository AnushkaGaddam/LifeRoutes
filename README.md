# LifeRoutes
Smart Ambulance Management System
Key Features:
LifeRoute intelligently recommends the most suitable hospital based on real-time facility availability and distance.
The system automatically detects the userâ€™s location and fetches nearby hospitals using OpenStreetMap.
It calculates a readiness score for each hospital based on ICU beds, ventilators, trauma care, maternity facilities, and specialist availability.
The AI Voice Assistant supports multiple languages including English, Telugu, and Hindi for emergency detection.
LensAI analyzes speech and camera input to classify emergency severity as Stable, Moderate, or Critical.
The application provides real-time route mapping using OSRM and displays the optimized path on an interactive map.
The system sends instant alerts to the hospital dashboard with patient details and ETA information.
A traffic prediction module simulates heavy and moderate traffic zones and adjusts ambulance movement accordingly.
The platform triggers early alerts to traffic control authorities to clear congestion before ambulance arrival.
The hospital dashboard prepares medical staff in advance based on real-time emergency notifications and estimated arrival time.
User selects emergency type or uses AI Voice or LensAI.
